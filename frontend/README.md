# Frontend Applications

This directory contains client-side web and mobile apps for the StadiumOS platform:

## Directory Index
- `ops-dashboard/`: The primary React 19 web application for the Operations Command Center. Runs Vite, Tailwind CSS, Material UI, and Recharts.

## Core Guidelines
- Follow Material Design 3 guidelines.
- Target WCAG 2.2 AA accessibility scores.
- Maintain strict typing using TypeScript configurations.
